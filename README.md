### ðŸ“¦ Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/Dhaboav/VoteApp.git
    ```

2. **Install dependencies with Poetry:**

    ```bash
    poetry install
    ```

2. **Run the application:**

    ```bash
    poetry run start
    ```

3. **Copy environment file:**

    ```bash
    copy .env.example .env
    ```

4. **Generate Secret Key:**

    ```bash
    openssl rand -hex 32
    ```
